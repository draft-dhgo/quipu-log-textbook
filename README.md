# The Quipu-Log Book

📖 **Read it: https://draft-dhgo.github.io/quipu-log-textbook/**

English | [한국어](README.ko.md)

A textbook for understanding [Quipu-Log](https://github.com/draft-dhgo/Quipu-Log) — a tamper-evident audit log — not as a list of features but as **concepts**. It teaches the storage engine and the security model by crossing from databases you already know to the filesystem internals you don’t.

- **8 parts, 35 chapters** — starting point / filesystem basics / the append-only log / re-creating DB guarantees on files / integrity / confidentiality / the write & read paths / distribution & operations
- Zero external dependencies, fully self-contained. Open `index.html` in a browser.
- Bilingual: English at the root, Korean under [`/ko/`](https://draft-dhgo.github.io/quipu-log-textbook/ko/).

## Build

Chapter bodies live in `bodies/<lang>/NN-slug.html`; the shared shell, navigation, and table of contents are generated by `build.py`.

```sh
python3 build.py   # manifest.json + bodies/{en,ko}/ → English at /, Korean at /ko/
```
